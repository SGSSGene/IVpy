# Introduction

**IV2py** provides python bindings for **IVio**.
The goal is to cover typical functionality for **bioinformatics**.

## Support

 - **fasta** (rw) + **gzip** (rw)
 - **sam** (rw)

(description - rw: read/write, ro: read-only)

**Note**: Currently only Linux is supported.
